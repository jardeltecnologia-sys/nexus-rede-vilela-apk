# Nexus Assistente Virtual APK

APK Android WebView premium para o assistente virtual Nexus.

## URL correta

```text
https://www.redevilela.com/
```

## O que este APK faz

Este aplicativo abre exclusivamente o assistente virtual Nexus em `www.redevilela.com`.

Não é o APK do portal de notícias `www.redevilela.com.br`.

## Recursos

- WebView Android nativa.
- Permissão de microfone para uso de voz no assistente.
- JavaScript, DOM Storage, cookies e mídia habilitados.
- Splash com vídeo enviado.
- Ícone premium Nexus.
- Barra inferior simples: Assistente, Microfone e Recarregar.
- Tela offline.
- Fullscreen.
- Workflow GitHub Actions para gerar APK automaticamente.
